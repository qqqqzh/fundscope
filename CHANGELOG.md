# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-24

### Added

- 初版项目文档，明确产品定位、运行方式与版本边界
- README 页面截图资源，用于 GitHub 项目展示
- Next.js 前端基础结构，包括仪表盘、基金筛选、基金详情、K 线学院、自选、持仓和投资模拟入口
- FastAPI 后端数据接口，用于基金、ETF、指数、净值和持仓相关数据聚合
- 自选列表本地持久化能力，以及基金详情页联动分析能力
- 持仓管理能力，支持本地录入持有金额、收益、仓位与类型分布查看

### Changed

- 将项目定位进一步收敛为“面向投资小白的学习与观察工具”
- 移除组合分析页面及其相关入口
- 将投资模拟页面改为待开发占位页，避免继续暴露未定型的模拟内容
- 将仓库从默认模板项目整理为可公开说明的首个版本结构

### Notes

- 当前版本适合作为个人研究和学习工具的公开起点
- 自选和持仓数据仍保存在浏览器本地
- 投资模拟功能将在后续版本中再补充
